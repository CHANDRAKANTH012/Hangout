Design a world-class, production-grade UI/UX for a modern web platform called “Hangout” — a real-time, location-based social meetup platform where users can discover nearby hangouts, create spontaneous events, join people around them, and chat in immersive live rooms.

This should NOT look like:

- a SaaS dashboard
- a generic glassmorphism template
- a crypto website
- a cloned dribbble landing page
- a basic social media app

This should feel like:
a premium consumer product built by elite product designers.

Think:
Linear × Arc Browser × Discord × Airbnb Experiences × Snapchat Map × Apple motion design × modern Framer award websites.

The design language should feel:

- emotional
- cinematic
- immersive
- youthful
- premium
- tactile
- modern
- socially alive

The platform should feel like:
“Friday evening energy.”
Spontaneous.
Warm.
Human.
Alive.

BACKGROUND SYSTEM (VERY IMPORTANT)

ALL pages should share a premium interactive background system.

Base:

- soft white / warm off-white textured background

Overlay:

- tiny black dotted grid system across the page
- inspired by Stitch / Linear / Figma aesthetic
- very subtle and elegant

Mouse proximity effect:

- dots near cursor become darker and slightly larger
- smooth radial interaction
- organic interpolation
- no hard spotlight

Ambient layer:

- extremely blurred warm gradient blobs
- low opacity
- slow movement
- coral / orange / amber / muted blue
- subtle atmospheric depth

The background should feel alive but never distracting.
